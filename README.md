# 家の味 - House History Search Fan Project

This is a **React frontend + Java backend API project** inspired by the horror game *家の味* (Taste of Home).  

The project replicates the in-game search page where the main character and their friend can check whether a house has a history of crime by entering the address.

## Architecture

- **Frontend:** React + TypeScript
- **Backend:** Java Spring Boot API
- **Styling:** CSS

## Features

- Search house addresses for crime history
- Interactive UI similar to the in-game experience
- React frontend communicates with Java API backend
